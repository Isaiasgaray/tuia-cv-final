# Trabajo práctico final - CV

Trabajo práctico final para la materia Visión por computadora de la Tecnicatura Universitaria en Inteligencia Artificial

# Requisitos

- Python 3.10 o superior

# Pasos para correr el código

## Crear el entorno virtual en el directorio root del repositorio

```sh
$ python -m venv venv
```

## Instalar las dependencias para el proyecto

```sh
$ pip install -r requirements.txt
```

## Activar venv

```sh
$ source venv/bin/activate
```

## Aplicación de gradio

Para correr la versión 1

```sh
$ python src/app_search_v1.py
```

Versión de la etapa 2 con selector de modelo

```sh
$ python src/app_search_v2.py
```
